Web Designer

Web Design refers to the designs of websites that are displayed on the internet. It usually refers to the user experience aspects of website development rather than software developer...
A web designer works on the appearance layout and in some cases, content of a website.

What Web-designers do ?

Web design identifies the goals of a website or webpage and promotes accessibility for all potential users. This process involves organizing content and images across a series of pages, integrating applications and other interactive elements. The professionals who perfore this process are called web designers and their job includes following duties:

  • Selecting easy-to-read fonts
  • Choosing attractive color schemes that also enable easy-to-read fonts
  • Implementing a brand's identity into the colors, fonts and layout
  • Creating a map of the website's structure to ensure intuitive navigation
  • Placing images, logos, text, videos, applications an other elements
  • Using coding languages, such as HTML and CSS, to create layouts and to style pages
  • Making optimized versions of websites and pages both for desktop and mobile viewing
